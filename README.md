## Installation

#### Expo Installation - Manual
You can use [Expo](https://expo.io/) to install :

```console
memefy@memefy:~$ git clone https://github.com/ahdhani/Memefier
```
```console
memefy@memefy:~$ expo install
```

```console
memefy@memefy:~$ expo start
```
